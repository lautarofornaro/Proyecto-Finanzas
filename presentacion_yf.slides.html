API yahoo finance¶

  • Usaremos la libreria de python yfinance, la cual nos facilita el trabajo de trabajar con la API de Yahoo finance
  • De dicha libreria podemos descargar informacion de los activos financieros que figuran en dicha pagina
  • Para este ejemplo en particular usaremos el activo de michrosoft o "MSFT"

Otras aplicaciones de la libreria yfinance¶

Algunas funcionalidades de la libreria

  • msft = yf.Ticker("MSFT")
  • msft.info, para ver la informacion del activo
  • msft.history(period="max"), para ver el historico del activo
  • msft.actions, para ver las acciones (mas que nada dividendos y splits)
  • msft.calendar, el calendario de dicho activo
